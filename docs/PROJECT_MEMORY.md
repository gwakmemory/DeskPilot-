# Brand Identity

DeskPilot is a workspace-first brand.

We help people create smarter, more comfortable, and more productive workspaces.

Products are tools—not the final goal.

(DeskPilot는 제품을 판매하는 브랜드가 아니라 더 나은 업무 공간을 만드는 브랜드다.)
